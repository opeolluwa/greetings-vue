## Custom

With the following rules you get a nice monospaced purple-colored dynamic greeting,
the possibilities are end endless

```css
.vue__greetings {
  font-size: 60% text-transform;
  color: #660066;
  display: inline;
  font-family: monospace;
  font-weight: bolder;
  padding: 3px;
  width: 45%;
}
```
