# Integration

Quicly integrate vue-greetings into your projects

# Register as a global component

import Vue from "vue";
import VueGreeting from "VueGreeting";
Vue.component("vue-greeting", VueGreeting);

then in vue components

<template>
  <div>
    <vue-greetings />
  </div>
</template>

# Register as Local Component

<template>
  <div>
    <vue-greetings />
  </div>
</template>

<script>
  import VueGreetings from "vue-greetings";
  export default {
    components: {
      VueGreetings,
    },
  };
</script>
